HaskellMinesweeper
==================

#Haskell Project 2014

  - Minesweeper game implementation
  - Graphical User Interface
  - Quickchecks
  - AI Solver

### Usage
```sh
$ git clone https://github.com/ghodgins/HaskellMinesweeper.git
$ cd HaskellMinesweeper/src
$ ghc interface.hs
$ ./interface
```
